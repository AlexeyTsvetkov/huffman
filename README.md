# Huffman encoder

File compression using Huffman 
[algorithm](https://en.wikipedia.org/wiki/Huffman_coding).

# Build

Run

```bash
make
```

# Usage

```bash
arj [-c|-d] -i INPUT_FILE -o OUTPUT_FILE
```

Parameters:
* -c -- compress
* -d -- decompress 
* -i -- input file path
* -o -- output file path
